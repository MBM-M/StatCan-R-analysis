# Canada's Economic Pulse

**Are Canadians keeping up? Wages, Housing, and Employment (2014–2024)**

An R-based analysis of Statistics Canada open data examining the relationship between wage growth, shelter costs, and employment trends across Canadian provinces.

---

## Overview

This project pulls data directly from the Statistics Canada Web Data Service (WDS) API, cleans it using tidyverse tools, and produces a reproducible HTML report analyzing housing affordability and post-pandemic labour market recovery.

**Key questions:**
- Has wage growth kept pace with shelter cost increases since 2014?
- Which provinces face the steepest housing affordability pressures?
- Which industries have recovered from COVID-19, and which haven't?

## Data Sources

All data is pulled from Statistics Canada open data tables:

| Table | Description |
|-------|-------------|
| [14-10-0023-01](https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=1410002301) | Employment by industry (seasonally adjusted) |
| [18-10-0005-01](https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=1810000501) | Consumer Price Index — All-items and shelter component, by province |
| [14-10-0064-01](https://www150.statcan.gc.ca/t1/tbl1/en/tv.action?pid=1410006401) | Average hourly wages by province and job type |

## Project Structure

```

├── fetch_data.R           # Downloads CSVs from StatCan WDS API
├── clean_data.R           # Cleans, filters, and merges datasets
├── 01_analysis_filled.Rmd # R Markdown report with charts and findings
├── 01_analysis_filled.html# Knitted HTML output
├── raw/                   # Raw CSVs (gitignored, regenerated by fetch_data.R)
└── processed/             # Cleaned analysis-ready CSVs (gitignored)
```

## How to Run

**1. Install required packages** (one-time setup)

```r
install.packages(c(
  "httr", "jsonlite", "dplyr", "tidyr", "readr",
  "lubridate", "stringr", "ggplot2", "scales",
  "forcats", "rmarkdown"
))
```

**2. Fetch and clean the data**

```r
source("fetch_data.R")   # Downloads ~500 MB of raw CSVs from StatCan
source("clean_data.R")   # Produces analysis-ready CSVs in processed/
```

**3. Knit the report**

Open `01_analysis_filled.Rmd` in RStudio and click **Knit**, or run:

```r
rmarkdown::render("01_analysis_filled.Rmd")
```

## Key Findings

- **Nationally, wages marginally outpaced shelter costs** by 2024 — wage index 139.8 vs. shelter index 137.9 (both 2014 = 100). The national average hides significant regional disparities.
- **Prairie provinces face the steepest affordability gaps.** Manitoba (+10.6 index points), Alberta (+9.7), and Saskatchewan (+5.0) saw shelter costs rise faster than wages.
- **Knowledge-economy and public sector jobs led post-COVID employment growth:** Professional, Scientific and Technical Services (+27.0%), Public Administration (+24.8%), and Finance and Insurance (+19.3%).
- **Service-sector jobs have not fully recovered.** Agriculture (-21.1%), Business & Support Services (-7.2%), and Accommodation & Food Services (-6.5%) remain below 2019 employment levels.
